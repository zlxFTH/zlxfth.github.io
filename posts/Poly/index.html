拉格朗日插值

(n+1)(n + 1)(n+1) 个 xxx 坐标不同的点能够唯一确定一个 nnn 次多项式。设多项式为 f(x)f(x)f(x)，第 iii 个点的坐标为 (xi,yi)(x_i, y_i)(xi​,yi​)，那么该多项式在 x=kx = kx=k 处的取值为 f(k)=∑i=0nyi∏i≠jk−xjxi−xjf(k) = \sum_{i = 0}^n y_i\prod_{i\ne j}\dfrac{k - x_j}{x_i - x_j}f(k)=∑i=0n​yi​∏i=j​xi​−xj​k−xj​​。暴力计算时间复杂度 O(n2)O(n^2)O(n2)。

如果 xxx 取值连续，分子可以写成前后缀积的形式，分母可以写成阶乘，可以 O(n)O(n)O(n) 计算。

1k+2k+⋯nk1^k + 2^k +\cdots n^k1k+2k+⋯nk 是关于 nnn 的 kkk 次多项式。

有一维值域特别大（比如说 10910^9109），基础 DP 式子很好推，那么可以考虑拉插。

答案可能是分段函数，每一段均为一个多项式，这时候采取分段插值的方式，因为总段数是 O(n)O(n)O(n) 的。（常见形式是题目里面出现了 [li,ri][l_i, r_i][li​,ri​] 这种限制，例如联合省选 2022 填树、NOI 2019 机器人）。注意前面的段可能对后面的段有贡献，一个 trick 是把前面的贡献放在 000 位置， [1,r−l+1][1, r - l + 1][1,r−l+1] 进行计算。

考虑将多项式用点值表示，那么多项式卷积就是把对应 xxx 位置上的值相乘，那么可以在 O(n)O(n)O(n) 计算多项式卷积了。但是系数和点值的相互转换都是 O(n2)O(n^2)O(n2) 的。

FFT 选用特殊的点值 xxx 来实现系数和点值的快速转换。多项式卷积后的次数为 n+mn + mn+m，求出最小的 n′=2k>n+mn' = 2^k > n + mn′=2k>n+m，令 n←n′n\leftarrow n'n←n′，把多项式的次数都补全到 n−1n - 1n−1 次。定义单位根 wnk=cos⁡2πkn+sin⁡2πkniw_n^k = \cos\dfrac{2\pi k}{n} + \sin\dfrac{2\pi k}{n}iwnk​=cosn2πk​+sinn2πk​i，形象一点 wnkw_n^kwnk​ 就是将复数平面上的单位圆分成了 nnn 份，然后从 1+0i1 + 0i1+0i 开始旋转 kkk 份得到的点。FFT 取 nnn 个 xxx，这 nnn 个 xxx 分别是 wn0,wn1,⋯wnn−1w_n^0, w_n^1,\cdots w_n^{n - 1}wn0​,wn1​,⋯wnn−1​。

单位根的性质： wnn=1,wnk=w2n2k,w2nn+k=−w2nk,wni+j=wni⋅wnjw_n^n = 1, w_n^k = w_{2n}^{2k}, w_{2n}^{n + k} = -w_{2n}^{k}, w_n^{i + j} = w_n^i\cdot w_n^jwnn​=1,wnk​=w2n2k​,w2nn+k​=−w2nk​,wni+j​=wni​⋅wnj​，证明显然。

递归常数太大了，需要使用蝴蝶变换优化。观察多项式系数序列通过不断递归最终的顺序，发现最终位置 iii 的值对应原序列 rir_iri​ 位置的值，其中 rir_iri​ 是 iii 的二进制翻转。O(n)O(n)O(n) 递推 rrr，首先 r0=0r_0 = 0r0​=0，如果 rir_iri​ 为偶数，ri=ri/2/2r_i = r_{i / 2}/2ri​=ri/2​/2，如果 rir_iri​ 为奇数，最高位还要补上一个 111。

那么我们并不需要递归，而是将多项式系数序列变换过后从底层往高层合并。对于系数转点值，枚举当前合并的两块长度 iii，枚举这两块的起始位置 jjj，当前要合并 j+kj + kj+k 和 i+j+ki + j + ki+j+k，那么 aj+k′=aj+k+w2ikaj+i+k,aj+i+k′=aj+k−w2ikaj+i+ka'_{j + k} = a_{j + k} + w_{2i}^k a_{j + i + k}, a'_{j + i + k} = a_{j + k} - w_{2i}^k a_{j + i + k}aj+k′​=aj+k​+w2ik​aj+i+k​,aj+i+k′​=aj+k​−w2ik​aj+i+k​。对于点值转系数，从一些理解角度中可以得知，只需要把单位根变成 wnkw_n^kwnk​ 的倒数 1wnk=cos⁡2πkn−sin⁡2πkn\dfrac{1}{w_n^{k}} = \cos\dfrac{2\pi k}{n} - \sin\dfrac{2\pi k}{n}wnk​1​=cosn2πk​−sinn2πk​，再做一遍 FFT，最后对每一项除以长度 nnn 得到答案。

实现细节：空间开 444 倍，因为 2k>n+m2^k > n + m2k>n+m 一定不会超过 nnn 的四倍。逆变换最后除以 nnn 之后加 0.50.50.5 保障精度。
