NOI2014 购票

发表于 2023-03-29 分类于 Solutions

Description

给定一棵 nnn 个点以 111 为根的树，uuu 和其父亲 F(u)F(u)F(u) 的距离为 S(u)S(u)S(u)，点 uuu 能够一步走到祖先 vvv，当且仅当 uuu 到 vvv 的距离不超过 L(u)L(u)L(u)，代价为 dis(u,v)×P(u)+Q(u)dis(u, v)\times P(u) + Q(u)dis(u,v)×P(u)+Q(u)。对于每个点求走到 111 的最小代价。

n≤2×105,0≤P(u)≤106,0≤Q(u)≤1012,0≤S(u)≤L(u)≤2×1011n\le 2\times 10^5, 0\le P(u)\le 10^6, 0\le Q(u)\le 10^{12}, 0\le S(u)\le L(u)\le 2\times 10^{11}n≤2×105,0≤P(u)≤106,0≤Q(u)≤1012,0≤S(u)≤L(u)≤2×1011，任意点走到 111 的距离不超过 2×10112\times 10^{11}2×1011。

Solution

设 f(u)f(u)f(u) 表示 uuu 走到 111 的最小代价。令 D(u)D(u)D(u) 表示 uuu 到根的距离。

f(u)=min⁡f(v)+(D(u)−D(v))×P(u)+Q(u)f(u) = \min f(v) + (D(u) - D(v))\times P(u) + Q(u) f(u)=minf(v)+(D(u)−D(v))×P(u)+Q(u)

套路地进行斜率优化。

f(u)−Q(u)−D(u)×P(u)=−D(v)×P(u)+f(v)f(u) - Q(u) - D(u)\times P(u) = -D(v)\times P(u) + f(v) f(u)−Q(u)−D(u)×P(u)=−D(v)×P(u)+f(v)

横坐标不连续，考虑使用李超线段树。先倍增求出 uuu 一步能走到的范围，树剖+线段树套李超线段树维护可以做到 O(nlog⁡3n)O(n\log^3 n)O(nlog3n)。

瓶颈在于树剖，使用出栈 DFS 序优化的 Trick，DFS 序记录每个点的出栈时刻，假设当前点为 uuu，能够走到最远的点的为 vvv，发现 u→vu\to vu→v 路径上所有直线构成的李超线段树恰好是 [dfn(u),dfn(v)][dfn(u), dfn(v)][dfn(u),dfn(v)] 这个区间的李超线段树，因为在这个区间但是还没有被访问的点还没有加入。

去掉树剖直接上线段树套李超树即可。

时间复杂度 O(nlog⁡2n)O(n\log^2 n)O(nlog2n)，空间复杂度 O(nlog⁡n)O(n\log n)O(nlogn)。
