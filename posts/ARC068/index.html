C. X: Yet Another Die Game

一个骰子，初始的时候任意的一个面朝上，接下来每一轮可以翻转骰子到相邻的一面，并获得此面的得分（那一面的数值即是得分）。问至少要几轮才可以获得够 xxx 分。

所有数在 long long 范围内，x≤1015x\le 10^{15}x≤1015。

2s,256MB2s, 256MB2s,256MB

D. Card Eater

有一堆牌，每张牌上有一个数字。 每次可以取出其中 333 张，丢掉数字最大的和数字最小的牌，把中间那张再放回牌堆。 要求最后所有剩余牌上的数字互不相同，求最多能剩几张牌。

3≤n≤105,1≤ai≤1053\le n\le 10^5, 1\le a_i\le 10^53≤n≤105,1≤ai​≤105，nnn 为奇数。

2s,256MB2s, 256MB2s,256MB

对每个颜色考虑，如果出现次数 ≥3\ge 3≥3，肯定是不断拿三个颜色相同的，如果是奇数剩 111 张，否则剩 222 张。

考虑剩两张的颜色个数 xxx，每 222 个颜色可以进行一次操作使得都变成 111 张，最后如果 xxx 为奇数，随便和一堆做一下，因为 nnn 是奇数一定有解。

E. Snuke Line

从左往右有 m+1m + 1m+1 个车站（编号 0∼m0\sim m0∼m），nnn 种商品，第 iii 种在 [li,ri][l_i, r_i][li​,ri​] 范围内的车站出售。对于 d=1∼md = 1\sim md=1∼m，求只经过 ddd 的倍数的车站能买到多少种商品。

1≤n≤3×105,1≤m≤1051\le n\le 3\times 10^5, 1\le m\le 10^51≤n≤3×105,1≤m≤105

2s,256MB2s, 256MB2s,256MB

对于 [l,r][l, r][l,r]，所有 d≤r−l+1d\le r - l+ 1d≤r−l+1 一定会在这个区间里面出现至少一次，可以直接计算这一部分的贡献。

剩下的 ddd 最多会在这个区间里面出现一次，O(mln⁡m)O(m\ln m)O(mlnm) 枚举倍数用 BIT 维护即可。

时间复杂度 O(mln⁡mlog⁡m)O(m\ln m\log m)O(mlnmlogm)。

F. Solitaire

将 1−n1-n1−n 顺序加入双端队列（每次可加头可加尾），再删除（每次可删头可删尾），求有多少种删除序列，使得 111 是第 kkk 个被删的。

1≤k≤n≤20001\le k\le n\le 20001≤k≤n≤2000

2s,256MB2s, 256MB2s,256MB

简单 DP 都不会了。

原始序列一定是先递减到 111 再递增，可以观察到删掉 111 其中一边一定是删完了的，剩下了一个单调的序列，每次可以从头开始删也可以从尾开始删，所以 111 后面的贡献为 2n−k−12^{n - k - 1}2n−k−1。

考虑前面的序列一定能够划分成两个单调递减的序列同时满足，存在其中一个序列的最小值是大于未选数的最大值的。

题目要求你要一个 O(n2)O(n^2)O(n2) 的 DP，先设 f(i,j)f(i, j)f(i,j) 表示选了 iii 个数 A 序列最小值为 jjj（钦定 B 序列的最小值大于未选数的最大值）的方案数。

  • 当前值划给 A 序列：f(i,j)→f(i+1,k)(k<j)f(i, j)\to f(i + 1, k)(k < j)f(i,j)→f(i+1,k)(k<j)；
  • 当前值划给 B 序列：f(i,j)→f(i+1,j)f(i, j)\to f(i + 1, j)f(i,j)→f(i+1,j)；

时间复杂度 O(n2)O(n^2)O(n2)。
