ARC152C Tree and LCS

发表于 2023-03-22 更新于 2023-03-29 分类于 Solutions

Description

给定长度为 nnn 的序列 aaa，每次操作选择序列中的一项，设它的值为 sss，执行 ∀i∈[1,n],ai←2s−ai\forall i\in [1, n], a_i\leftarrow 2s - a_i∀i∈[1,n],ai​←2s−ai​，需要保证每次操作后的序列始终非负，最小化 aaa 序列的最大值。

2≤n≤2×105,1≤a1<a2<⋯<an≤1092\le n\le 2\times 10^5, 1\le a_1 < a_2 < \cdots < a_n\le 10^92≤n≤2×105,1≤a1​<a2​<⋯<an​≤109。

Solution

考虑一次操作的本质，在数轴上将所有点围绕其中的一个点作翻转。因此序列最大值和最小值的差始终不变 d=an−a1d = a_n - a_1d=an​−a1​。

根据差值不变始终为 ddd，可以不断操作最大值把所有点挪到很远的位置来消除非负的影响，然后只需要在模 ddd 的意义下考虑。

观察发现奇数次操作可以让 a1a_1a1​ 加上 (2ai−2a1)(2a_i - 2a_1)(2ai​−2a1​)，偶数次操作可以让 a1a_1a1​ 减去 (2ai−2a1)(2a_i - 2a_1)(2ai​−2a1​)，显然可以通过不断插入 i=1i = 1i=1 来消除正负号的影响。

设 bi=(2ai−2a1)b_i = (2a_i - 2a_1)bi​=(2ai​−2a1​)，问题转换成求 a1+∑bici(modd),ci∈Za_1 + \sum b_ic_i\pmod d, c_i\in \mathbb{Z}a1​+∑bi​ci​(modd),ci​∈Z 的最小值，根据裴蜀定理，对 gcd⁡(g,bi)\gcd(g, b_i)gcd(g,bi​) 取模即可。

最后可以通过翻转左端点从很远的位置挪回来，答案即为 a1 mod gcd⁡(g,bi)+da_1\bmod \gcd(g, b_i) + da1​modgcd(g,bi​)+d。
