Tricks & Notices

发表于 2023-05-25 更新于 2023-06-05 分类于 Notes

长期补充。

  1. 二分

    • 最大值最小/最小值最大。
    • 答案是一段合法的前缀/后缀，或者有单调性。

  2. 两个状态走到同一个状态之后它们的状态之后一定相同？

    • 尝试维护相同的状态集合，比如说是一个连续段。

  3. 每个状态往后跳到的状态都是确定了，需要支持快速跳 kkk 步？

    • 倍增。

  4. 见到中位数？一个数 xxx 恰好为中位数？

    • 二分答案转成 −1/1-1/1−1/1 判断一个和是否 ≥0\ge 0≥0。

  5. 对于区间 [l,r][l, r][l,r]，每个 1≤d≤r−l+11\le d\le r - l + 11≤d≤r−l+1 的倍数在区间中至少出现一次，d>r−l+1d > r - l + 1d>r−l+1 的倍数在区间中最多出现一次。

  6. 随机化哈希

    • 注意一定要使用 mt19937_64！只有 unsigned long long 范围才能保证正确性。
    • 区间并交关系的处理：被同一个集合的区间所包含的点归为一类 -> 随机化哈希。
    • 无序集合的处理。

  7. 卡牌游戏模型：每个位置有两个数，选出一个使得 max⁡−min⁡\max - \minmax−min 最小。

    • 枚举其中一个端点进行处理。

贪心

  1. 线段覆盖问题
    • 选择最少的线段覆盖整个区间：左端点从小到大排序，右端点从大到小排序，贪心选能拓展最远的。
    • 选择最多的不重的线段：
    • 选最少的点覆盖所有线段：右端点从小到大排序，贪心选右端点。

树

  1. 从根往叶子考虑和从叶子往根考虑都想一下，从叶子往根考虑一般都是尝试搞定子树的子问题向上拓展。
  2. 维护树上邻域信息必死，父亲只有一个，转成维护子树信息再把父亲加进来。

图论

  1. 判环
    • 有向图 bfs：拓扑排序，如果入队点数不为 nnn 则有环。
  2. 网络流
    • 网格图转二分图，行列建点。
    • 限制点权的网络流，拆成一个入点和一个出点。

排序

  1. 给两个没有顺序的序列进行归并排序，相当于对前缀最大值进行归并排序，非前缀最大值总是在一段一段地跟在它前面最近的前缀最大值后面。
  2. 会对 01 序列排序的做法了，尝试类似中位数那样二分一个阈值？
  3. 冒泡排序交换次数就是逆序对个数，显然如果有两个位置已经确定并且构成逆序对，那么交换则更优。

二进制

  1. 二进制加法器 nnn 次给某一位加上 111 暴力进位是均摊 O(n)O(n)O(n) 的。
  2. 二进制压位成 2k2^k2k 的套路，kkk 一般是 4,8,16,32,64,1284, 8, 16, 32, 64, 1284,8,16,32,64,128。

DS

  1. 线段树优化建图

    • 连边是区间，建立正反两棵线段树，一棵儿子向父亲连边，一棵父亲向儿子连边，最好动态开点，因为区间向区间连边需要建立虚点，一个技巧是将两棵线段树的叶子节点都设成原图上的点。

    • 一个模板题 CF786B

  2. 位置难以维护尝试从值域入手？

    • 主席树上二分。

  3. O(n)O(n)O(n) 枚举前后缀划分然后左右答案可加，左右答案要 polylog(n)\text{polylog}(n)polylog(n) 计算？

    • 尝试能否利用单调性，每个前缀/后缀指针扫一遍算出答案去一个 log⁡\loglog。CF1837F

  4. 数颜色

    • 扫描线。
    • 维护前驱/后继，转换成只统计每个颜色在区间内第一个/最后一个。

DP

  1. 最简单的状态定义：你需要什么你就往状态里面塞什么。
  2. 划分 DP
    • 钦定选了的最靠右的状态归属于某个序列，显然状态要记录最靠右的是哪个。

串串

  1. 循环移位 kkk 次之后和原串 sss 相同   ⟺  \iff⟺ k ∣ ∣s∣k\ |\ |s|k ∣ ∣s∣ 并且 s1⋯ks_{1\cdots k}s1⋯k​ 是 sss 循环节。

博一博

  1. 每次给两个数减去相同的值，把原集合划分成两个和相同的集合对称取，后手必胜。

猜结论！

  1. 如果一个序列上我对 [1,n][1, n][1,n] 想到了某种贪心，并且猜到了和 1,n1, n1,n 这两个位置有关的结论，但是中间假了，那么尝试把 [1,n][1, n][1,n] 拓展到 [l,r][l, r][l,r]，对所有区间的情况取 max⁡/min⁡\max/\minmax/min。
  2. 猜测合法的取值是一个区间？那么只需要考虑区间的左端点和右端点了。
  3. 直接猜局部最优能导致全局最优？
  4. 敢于去打表找规律。
  5. 敢于去分析样例。

一些题目中巧妙的转化

  1. CF1809E：两杯水倒来倒去，但总量不变！
  2. P8500：只关注逆序对数量，只填出现过的数一定有最优解！
  3. AGC039B：相邻两个点所属集合编号绝对值之差为 111，任意两点之间限制转换成最短路问题。
  4. AGC039C：二进制循环移位，每次把最低位挪到最高位再取反，直接在右边拼一个原串取反的串。
