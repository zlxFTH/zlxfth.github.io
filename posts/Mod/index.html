阶与原根

阶

满足 an≡1(modp)a^n\equiv 1\pmod pan≡1(modp) 的最小正整数 nnn 称为 aaa 模 ppp 的阶，记为 δp(a)\delta_p(a)δp​(a)。

当 gcd⁡(a,p)=1\gcd(a, p) = 1gcd(a,p)=1 时有 aφ(p)≡1(modp)a^{\varphi(p)}\equiv 1\pmod paφ(p)≡1(modp)，所以 δp(a)≤φ(p)\delta_p(a)\le \varphi(p)δp​(a)≤φ(p)。

性质：

  • a,a2,⋯ ,aδp(a)a, a^2, \cdots, a^{\delta_p(a)}a,a2,⋯,aδp​(a) 互不同余。
  • 如 an≡1(modp)a^n\equiv 1\pmod pan≡1(modp)，则 δp(a)∣n\delta_p(a)|nδp​(a)∣n。
  • 若 gcd⁡(a,p)=gcd⁡(b,p)=1\gcd(a, p) = \gcd(b, p) = 1gcd(a,p)=gcd(b,p)=1，满足 δp(ab)=δp(a)δp(b)  ⟺  gcd⁡(δp(a),δp(b))=1\delta_{p}(ab) = \delta_p(a)\delta_p(b)\iff \gcd(\delta_p(a), \delta_p(b)) = 1δp​(ab)=δp​(a)δp​(b)⟺gcd(δp​(a),δp​(b))=1。
  • 若 gcd⁡(a,p)=1\gcd(a, p) = 1gcd(a,p)=1，则 δp(ak)=δp(a)gcd⁡(k,δp(a))\delta_p(a^k) = \frac{\delta_p(a)}{\gcd(k, \delta_p(a))}δp​(ak)=gcd(k,δp​(a))δp​(a)​。

原根

若 gcd⁡(a,p)=1\gcd(a, p) = 1gcd(a,p)=1 且 δp(a)=φ(p)\delta_p(a) = \varphi(p)δp​(a)=φ(p)，则称 aaa 为模 ppp 的原根。

原根判定定理：若 p≥3p\ge 3p≥3 且 gcd⁡(a,p)=1\gcd(a, p) = 1gcd(a,p)=1，则 [a is primitive root]=∀ primefactor P∈φ(p),aφ(p)P≢1(modp)[a\ is\ primitive\ root] = \forall\ prime factor\ P\in \varphi(p),a^{\frac{\varphi(p)}{P}} \not\equiv 1\pmod p[a is primitive root]=∀ primefactor P∈φ(p),aPφ(p)​≡1(modp)。

原根个数：若 ppp 有原根，则原根数量为 φ(φ(p))\varphi(\varphi(p))φ(φ(p))。

原根大小：若 ppp 有原根，则最小原根 ≤p4\le \sqrt[4] p≤4p​，所以暴力找为 O(p4log⁡2n)O(\sqrt[4]p\log^2 n)O(4p​log2n)。

原根存在定理：ppp 有原根   ⟺  p=2,4,Pk,2Pk\iff p = 2, 4, P^k, 2P^k⟺p=2,4,Pk,2Pk，PPP 为奇质数。

BSGS

普通 BSGS

给定 a,b,pa, b, pa,b,p，求出最小的 xxx，使得 ax≡b(modp)a^x\equiv b\pmod pax≡b(modp)，需要保证 gcd⁡(a,p)=1\gcd(a, p) = 1gcd(a,p)=1。

将 xxx 分解成 Ap−B (1≤A≤p,0≤B≤p)A\sqrt p - B\ (1\le A\le \sqrt p, 0\le B\le \sqrt p)Ap​−B (1≤A≤p​,0≤B≤p​)，则 aAp≡baB(modp)a^{A\sqrt p}\equiv ba^B\pmod paAp​≡baB(modp)。

O(p)O(\sqrt p)O(p​) 预处理 baBba^BbaB，用 hash table 存储，O(p)O(\sqrt p)O(p​) 枚举 AAA 查询。

exBSGS

给定 a,b,pa, b, pa,b,p，求出最小的 xxx，使得 ax≡b(modp)a^x\equiv b\pmod pax≡b(modp)，不保证 gcd⁡(a,p)=1\gcd(a, p) = 1gcd(a,p)=1。

设 g=gcd⁡(a,p)g = \gcd(a, p)g=gcd(a,p)：

ag×ax−1≡bg(modpg)\frac{a}{g}\times a^{x - 1}\equiv \frac{b}{g}\pmod {\frac{p}{g}} ga​×ax−1≡gb​(modgp​)

直到 aaa 和 ppp 互质为止。记 c=∏agc = \prod \frac{a}{g}c=∏ga​，ddd 为除的次数。

c×ax−d≡b(modp)ax−d≡b×c−1(modp)\begin{aligned} c\times a^{x - d}&\equiv b\pmod p\\ a^{x - d}&\equiv b\times c^{-1}\pmod p\\ \end{aligned} c×ax−dax−d​≡b(modp)≡b×c−1(modp)​

转换成普通的 BSGS。

细节：

  • bbb 不能被 ggg 整除，直接返回无解。
  • 答案可能小于最终的 ddd，需要在中途特判 x=dx = dx=d。
  • 先特判 p=1p = 1p=1 或者 b=1b = 1b=1 的情况（注意 000^000 未定义）
  • 求逆元的时候不能快速幂，只能用 exgcd，因为仅仅保证了 gcd⁡(a,P)=1\gcd(a, P) = 1gcd(a,P)=1，不保证 PPP 为质数。

求底数

求 xa≡b(modp)x^a\equiv b\pmod pxa≡b(modp)。

要求满足 ppp 为质数，保证 ppp 有原根且能使用 BSGS。
